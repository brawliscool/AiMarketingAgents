# MarketAI Frontend

Responsive static frontend for an AI marketing-agent SaaS.

## Live site

GitHub Pages URL: `https://brawliscool.github.io/AiMarketingAgents/`

## Deployment

The site is ready for GitHub Pages. In **Settings → Pages**, set the source to **GitHub Actions** once if Pages has not already been enabled, then run the included Pages workflow.

## Design

- Dark purple SaaS landing page matching the supplied mockup
- Interactive AI-agent network
- Marketing analytics dashboard
- Responsive desktop, tablet, and mobile layouts
- Demo modal, feedback toasts, and dashboard interactions
