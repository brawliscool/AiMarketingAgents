# MarketAI Design Direction

## Design read

Dark B2B AI-marketing SaaS landing page for small businesses, using a premium technical language and an asymmetric, product-led layout.

## Taste Skill dials

- `DESIGN_VARIANCE: 7`
- `MOTION_INTENSITY: 6`
- `VISUAL_DENSITY: 5`

## Applied principles

- Split-screen hero with a two-line desktop headline
- One intentional violet accent palette across the page
- Trial details separated from the hero into their own strip
- A real interactive product preview instead of a decorative fake screenshot
- Consistent corner-radius and CTA systems
- Responsive section-specific mobile layouts
- Motivated reveal, orbit, and feedback motion
- Reduced-motion support
- Sample analytics clearly labeled as example data
- Concise, functional copy with consistent signup intent
